# onivim-theme-hybrid
The Hybrid theme for vim, ported to Oni...

Usage:
```
# In config.tsx:
  "ui.colorscheme": "hybrid_dark", #dark theme
  "ui.colorscheme": "hybrid_light", #light theme
```
